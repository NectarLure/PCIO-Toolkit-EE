# Private Excluded File List

本文件列出未放入 `public_expert_review_site/` 的私有材料类型及排除原因。公开版未复制以下材料，也未保留指向这些材料的有效链接。

| 排除材料 | 是否放入公开版 | 排除原因 |
|---|---:|---|
| `paper/` 论文工作目录 | 否 | 包含论文初稿、投稿准备内容或仍在修改的研究文本，不适合作为公开站点材料。 |
| `manuscript.md` 论文初稿 | 否 | 论文正文尚属内部写作材料，公开版仅保留研究简介与证据边界说明。 |
| `submission_readiness_report.md` | 否 | 属于内部投稿判断和审稿策略材料。 |
| `change_log.md` | 否 | 属于内部改稿记录。 |
| `remaining_weaknesses.md` | 否 | 属于内部投稿前不足清单。 |
| `final_risk_check.md` | 否 | 属于内部风险检查材料。 |
| `前期论文/` | 否 | 可能包含未公开文献、草稿或内部资料。 |
| `研究提案/` | 否 | 属于内部研究设计与计划材料。 |
| `案例数据/` | 否 | 可能包含企业案例原文或敏感组织信息。 |
| `调查问卷以及数据/` | 否 | 可能包含原始调查记录或行级答卷。 |
| 原始T0调查CSV文件 | 否 | 行级调查数据不适合公开发布。 |
| 真实T0原始数据 | 否 | 存在隐私和重识别风险。 |
| 企业案例原文 | 否 | 可能包含企业名称、业务背景、人员、设备或地址等敏感信息。 |
| 专家原始评价 | 否 | 当前公开版只保留空白评分表，不包含已填写评价。 |
| `outputs/` 中任何真实企业输出 | 否 | 可能包含工具生成的真实组织报告、风险、KPI或情景输出。 |
| 企业真实名称、员工信息、设备真实编号、车牌、客户、供应商、地址、罐区敏感信息 | 否 | 公开版不得暴露可识别组织或个人的信息。 |
| 内部专家评审方案原文 | 否 | 公开版仅提供空白表、填写说明和公开材料包；内部方案如需获取需联系作者。 |
| 内部过度声明检查与投稿风险文件 | 否 | 属于内部改稿控制材料，公开版只保留通用证据边界说明。 |
